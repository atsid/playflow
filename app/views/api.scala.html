@main("Playflow") {

Projects

METHOD URL DESCRIPTION RETURNS PARAMETERS
GET /api/projects lists all active projects JSON workItems url: 'state' (initial, in_process, completed)
GET /api/projects/:projectId gets one project JSON workItem
POST /api/projects creates a new project JSON workItem body: new project JSON
PUT /api/projects/:projectId updates a project JSON workItem body: project JSON to update
DELETE /api/projects/:projectId deletes project
POST /api/projects/:projectId/assign assigns a project to a factory (unassigns for null 'factoryId') JSON workItem url: 'factoryId', 'transitionerId', 'comment'


Work Items

METHOD URL DESCRIPTION RETURNS PARAMETERS
GET /api/workItems lists active workItems JSON workItems url: 'projectId', 'assigneeId', 'stationId', 'state' (initial, unassigned, in_process, processed, completed)
GET /api/workItems/:workItemId get single workItem JSON workItem
POST /api/workItems creates a new workItem JSON workItem body: new workItem JSON
PUT /api/workItems/:workItemId updates a workItem JSON workItem body: workItem JSON to update
DELETE /api/workItems/:workItemId deletes workItem
POST /api/workItems/:workItemId/assign assigns a workItem to a user (unassigns for null 'assigneeId') JSON workItem url: 'assigneeId', 'transitionerId', 'comment'
POST /api/workItems/:workItemId/process indicates that a workItem has been processed JSON workItem url: 'transitionerId', 'comment'
POST /api/workItems/:workItemId/next moves a workItem to the next station (out of all stations for null 'nextStationId') JSON workItem url: 'nextStationId', 'transitionerId', 'comment'


Work Item History

METHOD URL DESCRIPTION RETURNS PARAMETERS
GET /api/workItems/:workItemId/history gets history of workItem state transitions JSON workItemStateTransitions


Factories

METHOD URL DESCRIPTION RETURNS PARAMETERS
GET /api/factories lists active factories JSON factories
GET /api/factories/:factoryId get single factory JSON factory
POST /api/factories creates a new factory JSON factory body: new factory JSON
PUT /api/factories/:factoryId updates a factory JSON factory body: factory JSON to update
DELETE /api/factories/:factoryId deletes factory


Stations

METHOD URL DESCRIPTION RETURNS PARAMETERS
GET /api/stations lists active stations JSON stations url: 'factoryId', 'projectId', 'workerId'
GET /api/stations/:stationId get single station JSON station
POST /api/stations creates a new station JSON station body: new station JSON
PUT /api/stations/:stationId updates a station JSON station body: station JSON to update
DELETE /api/stations/:stationId deletes station


Users

METHOD URL DESCRIPTION RETURNS PARAMETERS
GET /api/users lists active users JSON users
GET /api/users/:userId get single user JSON user
POST /api/users creates a new user JSON user body: new user JSON
PUT /api/users/:userId updates a user JSON user body: user JSON to update
DELETE /api/users/:userId deletes user
}
